About The Model

KNN Regression predicts a target value by averaging outcomes of the nearest data points in feature space. For this project, the model was trained on student-related attributes using Student_performance_data.csv.

The training workflow tested K values from 1 to 15 and selected the best performer as the final model. That best trained model was saved as best_salary_model.pkl.

During live usage, the Flask backend loads this saved model and returns prediction results to this frontend via the /predict API endpoint in real time.
